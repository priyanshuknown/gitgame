# Unity GTA-Lite Demo Workshop

## Project Overview
This project is a Unity demo workshop designed to provide an introduction to game development concepts, focusing on the mechanics and design inspiration from the GTA-like games. The workshop will guide participants through building a simple yet engaging game.

## Setup Instructions
1. **Prerequisites**: Ensure you have Unity installed on your machine. Download the latest version from the [Unity Download Page](https://unity3d.com/get-unity/download).
2. **Clone the Repository**: Use the command below to clone the repository:
   ```bash
   git clone https://github.com/priyanshuknown/gitgame.git
   ```
3. **Open the Project**: Launch Unity Hub and open the cloned project.
4. **Install Required Packages**: Go to Window > Package Manager and install any required packages as listed in the project documentation.
5. **Build and Run**: Once everything is set up, press the play button to run the demo or build it for your desired platform.
